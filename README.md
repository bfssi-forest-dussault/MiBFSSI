# MiBFSSI
Pipeline to process BFSSI MiSeq output. WIP.

## Internal Instructions
1. Login to the Venture cluster
```bash
ssh venture
```
2. Activate the conda env
```bash
source activate MiBFSSI-env
```
3. Navigate to the PyCharm project folder
```bash
cd /home/brock/PycharmProjects/MiBFSSI/MiBFSSI
```
4. Run the program with `python pipeline.py`
